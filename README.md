# 📅 HTML Timetable Project

## 📖 About

This project demonstrates how to create timetables using the HTML `<table>` element. It includes two examples: a school timetable and a weekly study schedule. The project helps beginners understand rows, columns, table headers, and data cells while practicing clean HTML structure and improving table design skills.

## ✨ Features

* 📊 HTML Tables
* 🏫 School Timetable
* 📚 Weekly Study Schedule
* 🔢 Rows & Columns
* 📝 Table Headers (`<th>`)
* 📄 Table Data (`<td>`)
* 💻 Beginner-Friendly Code

## 🚀 Purpose

The purpose of this project is to practice creating organized tables in HTML and understand how the `<table>`, `<tr>`, `<th>`, and `<td>` tags work together to display structured information.
